This repository serves as a test for the organization members.

If you are an organization member:
1. Fork this repository
2. Clone your fork locally
3. Add a file titled `your-username.md` (e.g., `PeppermintSnow.md`)
4. Stage your changes
5. Commit your changes with a meaningful message (e.g., `feat: added file for PeppermintSnow`)
6. Push your local changes to your remote Fork
7. Create a pull request from your fork to `isko-d/test-repo` on branch `main`
